# Printed Christian Evangelism Media Types Reference

This document outlines various types of printed Christian evangelism materials that could be offered through the CEM distribution service.

## Traditional Gospel Materials

- **Gospel tracts** - Short pamphlets with salvation message (current offering)
- **Gospel booklets** - Longer form materials, typically 8-16 pages
- **Gospel of John booklets** - Standalone gospel book
- **New Testament copies** - Complete New Testament
- **Four Spiritual Laws booklets** - Classic evangelism tool
- **Romans Road tracts** - Scripture-based salvation presentation

## Topical/Outreach Materials

- **Testimony cards** - Personal conversion stories
- **Invitation cards** - For church events and services
- **Holiday-specific tracts** - Christmas, Easter, etc.
- **Apologetics tracts** - Answering common questions about faith
- **Discipleship materials** - For new believers
- **Bible verse cards/scripture cards** - Memorization and encouragement

## Visual Materials

- **Comic-style gospel presentations** - Visual storytelling format
- **Illustrated children's tracts** - Age-appropriate gospel materials
- **Poster-sized materials** - For display and outreach events
- **Door hangers with gospel message** - Neighborhood outreach
- **Business card-sized gospel cards** - Easy to carry and share

## Special Purpose

- **Prayer cards** - Topical prayers and scripture
- **Follow-up materials** - For after initial contact with seekers
- **Church welcome packets** - Information for visitors
- **Funeral/memorial inserts** - Gospel message for memorial services
- **Hospital/crisis ministry materials** - Comfort and hope focused

## Multi-purpose

- **Calendars with daily scripture** - Year-round gospel exposure
- **Bookmarks with gospel message** - Practical with evangelistic purpose
- **Stickers** - Especially for children's ministry

## Implementation Notes

Most of these materials would fit the existing workflow with different values in the media type field.

### Suggested Type Categories

For the CEM system, consider these broad categories:

- **Short tract** - 1-4 page materials (existing)
- **Booklet** - 8+ page materials
- **Full gospel** - Complete book of Bible
- **Card** - Business card or invitation size
- **Poster/visual** - Display materials
- **Children's material** - Age-appropriate content

All materials would still support:
- Multi-language translations
- Digital PDF viewing
- Press-ready PDF downloads
- Quantity-based ordering
- Language filtering
